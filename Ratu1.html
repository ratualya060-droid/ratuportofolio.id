Proyek

Proyek 1: Implementasi Jaringan Tersegmentasi dengan VLSM

IPB University | Pemenuhan proyek mata kuliah Administrasi Aplikasi Jaringan

Proyek ini berfokus pada desain dan implementasi jaringan tersegmentasi yang memanfaatkan VLSM untuk efisiensi pengalokasian IP. Saya mengonfigurasi static routing pada tiga router utama, mengatur subnet secara terstruktur, serta memastikan komunikasi antar-segmen berjalan lancar. Pengujian dilakukan melalui Cisco Packet Tracer dan perangkat router untuk memastikan performa dan akurasi konfigurasi. Proyek ini meningkatkan keterampilan saya dalam desain jaringan serta manajemen routing dasar.

Proyek 2: Sistem Monitoring Suhu, Kelembapan, dan Cahaya dengan Indikator LED & Tampilan LCD

IPB University | Pemenuhan proyek mata kuliah Teknologi Sensor dan Aktuator

Mengembangkan sistem monitoring menggunakan sensor suhu & kelembapan, sensor cahaya (LDR), LCD display, serta LED indikator. Sistem menampilkan data lingkungan secara real-time dan memberikan tanda bahaya otomatis melalui LED ketika nilai sensor melewati batas aman. Proyek ini memperkuat pemahaman teknologi sensor, rangkaian elektronik, dan integrasi perangkat berbasis mikrokontroler.
